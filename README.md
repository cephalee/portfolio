# Portfolio Personnel - Étudiant en Informatique

Bienvenue sur le dépôt de mon portfolio personnel.  
Ce projet est une application web moderne développée avec **React** et **Vite**, conçue pour présenter mon parcours académique, mes compétences en développement logiciel et mes projets réalisés.

## Technologies utilisées

- **React** (Vite.js)
- **CSS3** (Animations, Flexbox, Grilles responsives)
- **JavaScript** (ES6+)

## Structure du Projet

```bash
mon-portfolio/
├── src/
│   ├── assets/             # Icônes, images des projets et CV PDF
│   ├── Bienvenue.jsx       # Composant d'animation de texte (**Effet machine à écrire**)
│   ├── Portfolio.jsx       # **Composant principal**
│   ├── index.css           # Design global et animations
│   └── main.jsx            # Point d'entrée React
└── autres                  # Dépendances et scripts
```

## Fonctionnalités

- **Animations Dynamiques** : Effet de machine à écrire multilingue sur la page d'accueil.
- **Navigation Fluide** : Système de navigation fixe avec défilement doux (**smooth scroll**).
- **Cartes de Projets** : Présentation interactive avec effets au survol et liens directs vers les dépôts **GitHub**.
- **Responsive** : Entièrement adaptable (**Mobile / Tablette / Desktop**) grâce aux Media Queries.

## Installation Locale

**Cloner le projet**
```bash
git clone https://github.com/cephalee/mon-portfolio.git
cd mon-portfolio
```

**Installer les dépendances**
```bash
npm install
```

**Lancer le mode développement**
```bash
npm run dev
```
